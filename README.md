# Groupe Moyo-Congo

Site vitrine officiel du Groupe Moyo-Congo.

## Structure du groupe

Le Groupe Moyo-Congo regroupe plusieurs plateformes indépendantes :

- Moyo Business : https://business.moyo-congo.com
- Moyo Dating : https://dating.moyo-congo.com

## Fichiers inclus

- `robots.txt` : autorise l’indexation du site par les moteurs de recherche.
- `sitemap.xml` : déclare les URLs principales du groupe et de ses plateformes.
- `README.md` : documentation simple du projet.

## Déploiement

Ces fichiers doivent être placés à la racine du projet du site principal :

https://moyo-congo.com

Exemple :

```txt
/
├── index.html
├── robots.txt
├── sitemap.xml
└── README.md
```

## Objectif

Présenter clairement le Groupe Moyo-Congo comme la maison mère de ses différentes plateformes digitales.
